PivotTable.js Examples

PivotTable.js is an open-source Javascript Pivot Table (aka Pivot Grid, Pivot Chart, Cross-Tab) implementation with drag'n'drop functionality written by Nicolas Kruchten at Datacratic.

View-source on these examples to see what's going on under the hood!
